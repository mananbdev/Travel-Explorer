# 🌍 Travel Explorer

A responsive multi-section travel landing page built using **HTML5** and **CSS3**.  
This project demonstrates modern layout techniques, smooth navigation, card-based UI design, and responsive styling without using JavaScript.

---

## 🚀 Live Preview

https://travel-explorer-4504.vercel.app/
---

## 📌 Overview

Travel Explorer is a static frontend website structured into multiple sections to simulate a real-world travel landing page.

The focus of this project is:

- Clean UI structure
- Responsive layouts
- Modern CSS techniques
- Organized section-based design

No frameworks or libraries are used — everything is built from scratch.

---

## 🧩 Features

- 📌 Fixed navigation bar with gradient background
- 🌅 Full-screen hero section (100vh) with background image
- 🏝 Destination cards built using Flexbox
- 📸 Responsive image gallery using CSS Grid
- 💼 Styled travel packages pricing table
- 💌 Contact form with clean input styling
- 🎨 Hover animations and smooth transitions
- 📱 Responsive layout principles

---

## 🛠️ Tech Stack

- **HTML5** → Semantic page structure
- **CSS3** → Layout, styling, animations
- **Flexbox** → Card alignment and spacing
- **CSS Grid** → Responsive gallery layout
- **Google Fonts (Poppins)** → Typography
- **External image sources (Unsplash, etc.)** → Visual assets

No JavaScript or external frameworks are used.

## ⚙️ Section Breakdown

### 🧭 Navigation Bar
- Fixed at top using `position: fixed`
- Gradient background
- Smooth scrolling using `scroll-behavior: smooth`
- Hover scaling effect on links

### 🌅 Hero Section
- Full viewport height using `100vh`
- Background image with text overlay
- Call-to-action button linking to Destinations

### 🏖 Destinations Section
- Flexbox layout with wrapping
- Card-based UI
- Hover zoom animation on cards

### 🖼 Gallery Section
- CSS Grid with:
  `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));`
- Equal height images using `object-fit: cover`
- Rotation + scale animation on hover

### 💼 Packages Section
- Styled pricing table
- `border-collapse` for clean table design
- Consistent padding and spacing

### 📩 Contact Section
- Centered form using Flexbox
- Rounded input fields
- Styled button with hover transform

### 🦶 Footer
- Dark theme footer
- Copyright section
